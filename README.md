# ScalaParBiBit
[![License](http://img.shields.io/:License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0.html)
[![DOI:10.1007/s10586-021-03261-z](https://img.shields.io/badge/DOI-10.1007/s10586--021--03261--z-blue)](https://doi.org/10.1007/s10586-021-03261-z)

Parallel tool to accelerate the search of biclusters on binary datasets described in the paper
> B. B. Fraguela, D. Andrade,  J. González-Domínguez. [ScalaParBiBit: scaling the binary biclustering in distributed-memory systems](http://gac.udc.es/~basilio/papers/Fraguela21-ScalaParBiBit.pdf). Cluster Computing, 24(3), pp. 2249-2268. September 2021. ([DOI 10.1007/s10586-021-03261-z](http://dx.doi.org/10.1007/s10586-021-03261-z))

Please find the instructions for building and using the tool in the file RefManual.pdf
